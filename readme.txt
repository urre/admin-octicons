== Description ==

Use Octicons in wp-admin.

* [Github](https://github.com/urre/admin-octicons)

== Installation ==

1. Upload the `admin-octicons` folder to your `/wp-content/plugins/` directory
2. Activate the "Admin Octicons" plugin in your WordPress administration interface
3. Done!
